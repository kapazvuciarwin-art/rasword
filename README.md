# 日文單字練習程式

一個簡單的日文單字練習 Web 應用程式，使用 Flask + SQLite 建立。

## 功能

- **輸入單字**：可以單筆或批次輸入日文單字
- **練習題目**：隨機出題，提供 10 個選項讓你選擇正確的中文解釋
- **提示功能**：可查看常用短句作為提示
- **單字列表**：查看和管理已儲存的單字

## 資料庫欄位

- 日文詞
- 詞性
- 常用短句 1
- 常用短句 2
- 中文解釋
- 日文字典等級 (JLPT)

## 安裝與執行

```bash
# 安裝依賴
pip install -r requirements.txt

# 執行程式
python app.py
```

程式會在 http://localhost:5000 啟動

## 批次輸入格式

使用 `|` 分隔各欄位，每行一個單字：

```
日文詞|詞性|常用短句1|常用短句2|中文解釋|等級
```

範例：
```
食べる|動詞|ご飯を食べる|朝ご飯を食べました|吃|N5
飲む|動詞|水を飲む|コーヒーを飲みます|喝|N5
```
